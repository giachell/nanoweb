About Nanoweb

Nanoweb is a Web search system by means a user can find relevant nanopublications for a specific information need.
Users can provide queries in natural language and the Nanoweb search engine returns a list of the relevant judged nanopublications for the given query.
The Nanoweb interface allows:


  • Full-text search over the nanopublications.
  • Cite nanopublications.
  • Access and visualize nanopublications.
  • Explore nanopublications and their relation network.
